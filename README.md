# Weather App

A modern weather application built using HTML, CSS, and JavaScript that provides real-time weather information, air quality data, sunrise/sunset timings, recent search history, dark mode, and a 3-day weather forecast.

## Screenshot


## Features
- Search weather by city name
- Press Enter to search
- Current temperature
- Feels like temperature
- Humidity
- Wind Speed
- Air Quality Index (AQI)
- Atmospheric Pressure
- Weather condition with icon and emoji
- Sunrise and sunset timings
- Live date and time display
- Dark mode / Light mode toggle
- Recent search history
- Click a city from history to search again
- Clear search history button
- 3-Day weather forecast

## Setup

1. Get a free API key from WeatherAPI.
2. Open `index.html`.
3. Replace:
```javascript
